### 二叉树的性质

1. 在二叉树的第`i`层至多有：$ 2^{i-1} $(`i >= 1`)
2. 深度为`k`的二叉树至多有 $  2^k-1 $节点(`k >= 1`)
3. 对于任意一个二叉树，如果其终端节点为：$ n_0 $，度为2的节点数为：$ n_2 $，则$ n_0 = n_2 + 1 $

### 问题:程序多连续运行几次出现：***`0xC0000005`***

- 多次运行程序中未对`malloc`分配的内存进行释放，必须进行释放，不然多次运行就会出现报错
